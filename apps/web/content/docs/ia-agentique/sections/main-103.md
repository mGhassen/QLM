Parce que les environnements sont instantanés et jetables, les équipes testent davantage, attendent moins et livrent plus souvent. Nous mesurons l'effet via les quatre indicateurs DORA, standard du secteur. Le graphique ci-contre montre le temps gagné par opération : les actions que les agents réalisent le plus, provisionner, rafraîchir et cloner, sont celles qui s'améliorent le plus.
