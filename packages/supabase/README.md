# Supabase - @guepard/supabase

This package is responsible for managing the Supabase client and various utilities related to Supabase.

Make sure the app installs the `@guepard/supabase` package before using it.

```json
{
    "name": "my-app",
    "dependencies": {
        "@guepard/supabase": "*"
    }
}
```
